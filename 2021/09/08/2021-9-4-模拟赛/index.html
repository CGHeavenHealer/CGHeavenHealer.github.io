# T1

[FJSC2021D9T1] 殖民主义

小水题

可以发现双方的最优策略是沿着 Spain 和 Portugal 两个点所在的链上，Spain 往下走，Portugal 往上走。原因：设 xxx 为 Portugal 控制的深度最浅的点，那么它可以控制住 siz[x]siz[x]siz[x] 的所有点，而 Spain 可以控制住剩下的点。

可以以 111 为根直接 DFS，记录一下 fa,sizfa,sizfa,siz，之后从 nnn 暴力跳 fafafa，找出 1−n1-n1−n 的链，之后用一个双指针从两侧扫描数组，交汇时即可统计答案。

注意如果最后一次移动是 rrr 移动的，并且移动次数为偶数，即轮到 Portugal，答案就多统计了一个，需要特判一下。

# T3

「JOI 2017 Final」足球

拆点 + 最短路

对于球，我们可以用 666 个状态描述它：

  • 1−41-41−4：向四个方向滚动；
  • 555：控球；
  • 666：不被控；

对于每种状态，我们对应连边：

  • 对于 1−41-41−4，可以向每个方向连一条代价为 aaa 的边，表示继续滚动；
  • 对于 1−41-41−4 到 666，可以连一条没有代价的边，表示球停止滚动；
  • 对于 555，向这个点周围四个点连代价为 bbb 的边，表示球被球员踢起来了；再向下一个点连一个代价为 ccc 的边，表示有一个球员带球；
  • 对于 666，向 555 连一条代价为 ccc 的边，表示有一个球员前来控球。

BFS 预处理出曼哈顿距离，按照上述情况连边，求最短路即可。

# T4

「BalticOI 2019 Day1」山谷

从出口到 rrr 的连通性不好维护，所以可以转换一下，先以 eee 为根树剖，如果 rrr 在删的那条边里 depdepdep 较大的点的子数里，说明从 rrr 到不了 eee 了。如果到不了 eee，子树内也没有商店，就无解了。

之后处理商店之间的距离：用 fuf_ufu​ 表示在 uuu 的子树中，离 uuu 最近的商店距离，可以得到转移式：fu=min⁡(fv+disu,v)f_{u}=\min(f_v+dis_{u,v})fu​=min(fv​+disu,v​)。

所以要求的其实就是链上的最小值，可以在 DFS 序上维护 fi−disif_i-dis_ifi​−disi​，这里的 disdisdis 指的是树上的前缀距离和。直接线段树即可。
