# T1

【2018.10 雅礼集训 day3】T1 u

一个特殊的二维差分

如果直接暴力修改，复杂度显然 Θ(n2m)\Theta(n^2m)Θ(n2m)，不可接受。

考虑优化这个过程：我们可以把暴力修改整个三角形改成对每一行或每一列差分一下，这样复杂度就降成了 Θ(nm)\Theta(nm)Θ(nm)，能拿到 616161 分的成绩了。

但是这样还是远远不能通过这道题的。

可以观察一下，斜着的线很有规律，我们可以先对列差分，在 (r,c),(r+l,c)(r,c),(r+l,c)(r,c),(r+l,c) 两个位置加上对应的权值，之后在斜线上差分，在 (r,c+1),(r+l,c+l+1)(r,c+1),(r+l,c+l+1)(r,c+1),(r+l,c+l+1) 的位置上打一个标记，之后累加答案时减掉斜线上的前缀和即可。如下图：

询问答案时直接用列的前缀和减掉斜线的差分即可。复杂度 Θ(m)\Theta(m)Θ(m)

# T3

【2018.10 雅礼集训 day4】T1 x

显然两个数如果不互质，必须要分到同一个集合里；我们可以用并查集维护这样的连通块。

设连通块个数为 tottottot，统计答案时，由于每个连通块可以分到集合 111，也可以分到集合 222，总的方案数应该是 2tot2^{tot}2tot。但是要减去全在集合 111 或全在集合 222 的情况，最终答案是 2tot−22^{tot}-22tot−2。

具体实现时，先处理出每一个数的因数和范围内的质数，再枚举整个序列和他们的因数，合并集合。

# T6

【2018.10 雅礼集训 day7】Reverse

最简单的思路：每个点像能到达的点连边，跑最短路；

可以有两种思路优化：线段树优化建图、 std::set

这里给出第二种的讲解。

可以发现，一个点能到达奇点还是偶点是固定的。

所以可以用两个 set 维护奇点和偶点，之后跑 BFS，每转移到一个点就从 set 中删掉他，复杂度是有保证的。
