# 简介

平衡树是一种二叉搜索树（BSTBSTBST）。

~~ 众所周知，~~ 二叉树上的操作与树高是成正比的。随机构成一棵BSTBSTBST 的期望树高是 $ log_{2} n $，但是最坏的情况树高会达到 $ n $。为了防止在二叉搜索树上的操作复杂度退化，平衡树便应运而生。

平衡树有很多种，像AVL，Treap，Splay，SBT，AVL，Treap，Splay，SBT，AVL，Treap，Splay，SBT，红黑树，包括STLSTLSTL 里的setsetset 和multisetmultisetmultiset 内部也是平衡树实现的，其中TreapTreapTreap&SplaySplaySplay 比较实用 （其实是 Renamoe 大佬告诉我其他的没有用）。本篇主要介绍 Treap。（其实是本蒟蒻只会TreapTreapTreap）

# Treap​

TreapTreapTreap 是一个合成词，由TreeTreeTree 和HeapHeapHeap 两个词合成。这也就意味着其有二者共同的性质 —— 既满足堆（任意节点的优先度都大于其子节点），又满足BSTBSTBST 的性质（任意节点左子树上的点的权值都小于根节点，任意节点右子树上的点的权值都大于根节点，其中序遍历为一个非严格单调递增的序列）。

由于TreapTreapTreap 要满足堆的性质，又要期望平衡，所以最简单的方法就是随机优先级，这样生成出的 $ Treap $ 是期望平衡的。

l,rl,rl,r 分别指左右儿子，vvv 指权值，datdatdat 指优先级，size,cntsize,cntsize,cnt 指子树大小及副本数（权值相同的点的个数）。

# 给权值求排名

有以下几种情况：

1. 该节点为空节点：则没有以vvv 为权值的点，检索失败。

2. 该节点值等于vvv：则其排名为sizp[l]+1siz_{p[l]} + 1sizp[l]​+1；

3. 该节点值小于vvv：查询左子树；

4. 该节点值大于vvv：查询右子树；

对于 3 和 4，加上左子树的大小以及节点的副本数即为答案。

# 给排名求权值

也是有以下几种情况：

1. 该节点为空节点：则其权值为 $ \infty $（其实不会出现）；

2. 该节点值左子树大小大于等于rankrankrank：则在其左子树里递归求解；

3. 该节点左子树大小 + 节点副本小于rankrankrank：该节点的权值即为答案；

4. 否则在右子树中查找 $rank - siz_{p [l]} - cnt_p $。

# 旋转

旋转应该是平衡树保持平衡的最基本操作了。

# 左旋

提当前节点的左儿子：

void zag(int &p) {
    int q = t[p].r;
    t[p].r = t[q].l;
    t[q].l = p;
    p = q;
    update(t[p].l);
    update(p);
}
# 右旋

提当前节点的右儿子：

void zig(int &p) {
    int q = t[p].l;
    t[p].l = t[q].r;
    t[q].r = p;
    p = q;
    update(t[p].r);
    update(p);
}

# 插入

插入一个权值为vvv 的点：

1. 若没有这个点，则在递归最深处新建一个节点；

2. 如果已经有了，则增加一个副本；

否则：

1.v<t[p].vv < t[p].vv<t[p].v：递归插入到左子树；不满足堆性质则右旋；

2.$ v \ge t [p].v$：递归插入到右子树；不满足堆性质则左旋；

void insert(int &p, int v) {
    if (p == 0) return p = New(v), void();
    if (v == t[p].v) {
        t[p].cnt++;
        update(p);
        return;
    }
    if (v < t[p].v) {
        insert(t[p].l, v);
        if (t[p].dat < t[t[p].l].dat) zig(p);
    } else {
        insert(t[p].r, v);
        if (t[p].dat < t[t[p].r].dat) zag(p);
    }
    update(p);
}

# 删除

删除一个权值为vvv 的点：

1. 若没有这个点，则返回；

2. 如果已经有了：

​ 1. 有多个：减少副本数；

​ 2. 有一个：

​ 1. 如果其左右儿子不为空：

​ 1. 如果右儿子为空，或不满足堆性质则右旋；

​ 2. 如果左儿子为空，或不满足堆性质则左旋；

​ 否则：没有该节点，p=0p = 0p=0；

否则：

1.v<t[p].vv < t[p].vv<t[p].v：递归到左子树；

2.$ v \ge t [p].v$：递归到右子树；

void remove(int &p, int v) {
    if (p == 0) return;
    if (v == t[p].v) {
        if (t[p].cnt > 1) {
            t[p].cnt--;
            update(p);
            return;
        }
        if (t[p].l || t[p].r) {
            if (t[p].r == 0 || t[t[p].l].dat > t[t[p].r].dat) {
                zig(p);
                remove(t[p].r, v);
            } else {
                zag(p);
                remove(t[p].l, v);
            }
            update(p);
        } else
            p = 0;
        return;
    }
    v < t[p].v ? remove(t[p].l, v) : remove(t[p].r, v);
    update(p);
}
