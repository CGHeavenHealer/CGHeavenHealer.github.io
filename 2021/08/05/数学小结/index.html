# 康托展开

广义康托展开：变进制数下的进制变换

康托展开：将任意十进制数转换为阶乘进制数

逆康托展开：将任意阶乘进制数转换为十进制数

构造双射变换： P→S!:xi=ranksuf(pi)P \to S! : x_i = rank_{suf}(p_i)P→S!:xi​=ranksuf​(pi​)

构造方法：从给定的排列的最高位开始扫，统计一下比这一位数小、且没使用过的数的个数，并把这个数标记为已使用（因为求的是排列，数与数之间不能重合），然后乘上该位的阶乘。

如果数字过多，可以用树状数组维护一下数字有没有使用过。

# 逆元

在剩余系中，加法、减法、乘法关于mod  p\mod pmodp 都是满足分配律的，即：

(a+b)mod  p=(amod  p)+(bmod  p)(a+b) \mod p = (a \mod p )+( b \mod p)(a+b)modp=(amodp)+(bmodp)

(a−b)mod  p=(amod  p)−(bmod  p)(a-b) \mod p = (a \mod p )-( b \mod p)(a−b)modp=(amodp)−(bmodp)

(a×b)mod  p=(amod  p)×(bmod  p)(a\times b) \mod p = (a \mod p) \times( b \mod p)(a×b)modp=(amodp)×(bmodp)

而对于除法，这样的法则是不适用的。

在实数域运算时，除以一个数等于乘以这个数的倒数；而在剩余系中，也可以类似的考虑，这样得到的 “倒数” 就是逆元：

在mod  p\mod pmodp 意义下，设 a−1a^{-1}a−1 为 aaa 的乘法逆元，则有：

a×a−1=1(mod  p)a \times a^{-1} = 1 (\mod p)a×a−1=1(modp)

这样，在处理完逆元后，就可以在剩余系中做除法运算了。
