# 基本原理

加法原理：做一件事，方法有 nnn 类，其中第 iii 类方法有 mim_imi​ 种实现方式，则完成这件事的方法有 ∑i=1nmi\sum_{i=1}^nm_i∑i=1n​mi​ 种。

乘法原理：做一件事，步骤有 nnn 个，其中第 iii 个步骤有 mim_imi​ 种实现方式，则完成这件事的方法有 ∏i=1nmi\prod_{i=1}^nm_i∏i=1n​mi​ 种。

减法原理：合法方案 = 总方案 - 非法方案（补集思想）

# 卡特兰数

Cat_n=\dfrac{C_{2n}^n}{n+1}=\sum_{i=0}^n Cat_i\times Cat_

实际意义：

  1. 从原点 (0,0)(0,0)(0,0) 走到 (n,n)(n,n)(n,n) ，只能向上或向右移动一个单位，求路线处于 y=xy=xy=x 这条直线之下的方案数
  2. 一个无穷大的栈，有 1−n1-n1−n 这几个数经过，合法的出栈序列
  3. nnn 个节点构成的二叉树数量
  4. n+3n + 3n+3 个点的多边形三角划分方案数
  5. nnn 个左括号和 nnn 个右括号构成的合法括号序列

# 斯特林数

第二类斯特林数：把 nnn 个数划分为 mmm 个集合的方案数（将 nnn 个不同的球放到 mmm 个无差别的盒子里，要求盒子非空的方案数）

公式：S(n,m) = \dfrac {(-1)^k \times C_m^k \times (m - k) ^ n}

递推式：考虑第 nnn 个元素的情况：

  • 若 n−1n - 1n−1 个元素构成了 m−1m - 1m−1 个集合，则第 nnn 个元素单独构成一个集合，方案数为 S(n,m−1)S(n, m - 1)S(n,m−1)；
  • 若 n−1n - 1n−1 个元素已经构成了 mmm 个集合，将第 nnn 个元素插入到任意一个集合，方案数为 m×S(n,m)m \times S(n, m)m×S(n,m)；

所以，第二类斯特林数的递推式为 S(n,m)=S(n,m−1)+m×S(n,m)S(n,m) = S(n, m - 1) + m \times S(n, m)S(n,m)=S(n,m−1)+m×S(n,m)。

应用：小球装箱

  1. 将 nnn 个不同的球放到 mmm 个无差别的盒子里，不允许盒子为空。

    答案：S(n,m)S(n, m)S(n,m)

  2. nnn 个不同的球，放入 mmm 个有区别的盒子，不允许盒子为空。

    答案：m!×S(n,m)m!\times S(n, m)m!×S(n,m) （盒子有区别，要乘上盒子的排列）

  3. nnn 个不同的球，放入 mmm 个无区别的盒子，允许盒子为空。

    答案：∑k=0mS(n,k)\sum_{k=0}^mS(n, k)∑k=0m​S(n,k) （枚举非空盒子的数目）
