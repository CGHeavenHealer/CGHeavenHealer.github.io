# 简介

莫队是一种优雅的暴力分块算法，由于其为国家队队长莫涛发明，故称为莫队。

与其他的分块不同，莫队不是讲序列分块，而是将询问按端点排序后分块，朴素的解决完第一个询问后，左右端点分别向下一个询问移动（把多出来的删掉，不够的加上去）。

# 答案

题干中让我们求概率：

我们设cnticnt_icnti​ 表示 $ [l,r] $ 内某种颜色的袜子出现的次数，则在区间 [l,r][l,r][l,r] 中取出两个相同颜色的袜子情况有：

C_{cnt_x}^2 = \dfrac {cnt_x * cnt_x-1} 2 $$; 在区间 $[l,r]$ 中取出两个袜子情况共有： $$ C_{len}^2 = \dfrac {len*(len-1)} 2 ,(len = r - l + 1)$$； 所以，概率为： $$ P = \dfrac {cnt_x*(cnt_x-1)} {len * (len - 1)} = \dfrac {cnt_x*(cnt_x-1)} {(r - l + 1)* (r - l)} = \dfrac {cnt _ x ^ 2 - cnt_x} {(r - l + 1)* (r - l)}$$； 即，抽到两个相同数字的概率为： $$ \sum_{i=1}^n \dfrac {cnt _ i ^ 2 - cnt_i} {(r - l + 1)* (r - l)} = \dfrac {\sum_{i=1}^n cnt _ i ^ 2 - cnt_i} {(r - l + 1)* (r - l)}$$； 而对于区间$[l,r]$，有： $$ \sum_{i=1}^n cnt_i = r - l + 1 $$； 则答案为： $$ ans=\dfrac{\sum_{i=1}^n cnt_i^2 - (r - l + 1)}{ (r - l + 1)*(r-l) }$$。 --- ##### $ \color{lightgreen} Code: $ ```c++ #include using namespace std; #define int long long #define ri register int const int N = 1e5 + 10; inline int read() { ri x = 0, f = 1; char ch = getchar(); for (; !isdigit(ch); ch = getchar()) if (ch == '-') f = -1; for (; isdigit(ch); ch = getchar()) x = (x << 1) + (x << 3) + (ch ^ 48); return f * x; } struct node { int l, r, id; } mo[N]; int L[N], R[N], ans[N], s[N], sum, a[N], lal = 1, lar, t; int gcd(int a, int b) { return b ?
