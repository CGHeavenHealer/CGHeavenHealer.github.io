# T1

0826 农夫约的假期 (shuru)

由题意可得，两个点之间的代价就是两个点的曼哈顿距离，但如果 n2n^2n2 直接求的话，时间、空间都不可接受。

但这道题中的横竖两个维度互不影响，可以把二维的做法压成一维，分别递推即可。

# T2

换根 DP

可以发现每次换根后，只有一条边和两个点受到了影响：换根前的贡献：(e[i].w−val[fa])×siz[v](e[i].w-val[fa])\times siz[v](e[i].w−val[fa])×siz[v]，换根后的贡献：(e[i].w−val[v])×(siz[1]−siz[v])(e[i].w-val[v])\times (siz[1]-siz[v])(e[i].w−val[v])×(siz[1]−siz[v])。

把贡献更改一下，然后回溯搜索即可。

# T3

有一个显然的结论：深度最小的 LCA 只能是 DFS 序离这个点最近的。

有了这个结论后，就很简单了，只要维护一个用来查前驱后继的数据结构，像平衡树、权值线段树，甚至 std::set 就行了。
