# 前言

最近发现自己大降智，CSP-S\text{CSP-S}CSP-S 一道题都切不了……

可能是颓的太多了，也可能是写 SB 数据结构写傻了，也可能是贺题贺的太多了……

看来确实有必要做点思维题、贪心、DP 练练自己的脑子。

CF1304C Air Conditioner

对于一个时刻 ttt，他能到的温度区间为 [m−t,m+t][m-t,m+t][m−t,m+t]。

大体思路：只需要判断所有区间在每一个时刻是否在值域区间内即可。

细节：每次更新值域区间 [l,r][l,r][l,r] 时，要注意它往下不能低于这个询问的 lll，往上不能高于这个询问的 rrr。

CF1385D a-Good String

字符串长度是 222 的整次幂，可以递归处理。每次比较一下取两边哪个更小即可。

和 这个题 差不多，都是要递归处理。

CF1542B Plus and Multiply
